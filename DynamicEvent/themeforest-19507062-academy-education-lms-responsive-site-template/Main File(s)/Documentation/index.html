A) HTML structure - top

This template created using Bootstrap framework. All the information within main content is nested within a div with id- name "wrapper". If you need more information you can read Bootstrap documentation on this link

1) Wrapper-Structure

    <div id="wapper">
    	<!--///////////// Mainmenu Begin /////////////-->
        <header id="quck-nav"> .... </div> 
        <header id="header"> .... </div> 
        	
        <!--///////////// Slider index page Begin /////////////-->
        <section id="banner"> .... </section>
        
        <!--///////////// Breadcrumb Begin /////////////-->
        <div class="breadcrumb"> .... </div>
        
        <!--///////////// Main Show Content Begin /////////////-->
        <section>
            <div class="container">
                <div class="row"> .... </div>
                <div class="row"> .... </div>
            </div> 
        </section>
        <!--///////////// Footer Begin /////////////-->
        <footer id="footer"> .... </footer>
    </div>
    <!-- //wrapper -->

 

2) Main-Structure

All of the information within the main content area is nested within a div with an id of #wrapper

    <!--///////////// Main Show Content Begin /////////////-->
    <section class="class">
        <div class="container">
            <div class="row">     
            	<div class=col-md-4">...</div>     
            	<div class=col-md-8">...</div>
            </div>
        </div>
    </section>
    <!-- //wrapper -->

 

3) Basic Grid HTML

For a simple two column layout, create a .row and add the appropriate number of .col-md-* columns. As this is a 12-column grid, each .col-md-* Col a number of those 12 columns and should always add up to 12 for each row (or the number of columns in the parent). Click here for more information.

    <div class="row">
    	<div class=col-md-4">...</div>
        <div class=col-md-8">...</div>
    </div>

 

B) CSS Files and Structure - top

This template includes a number of CSS files that are used for display the website. All CSS files, can be found in the Template's css/ folder and some plugins folder.

The CSS file :

  1. font-awesome.min.css - It inludes the complete set of icons used in our theme
  2. bootstrap.css - It includes the based on Bootstrap Framework from Twitter
  3. datepicker.css -It is used for styleable datepicker.
  4. owl.carousel.css - It used for slider
  5. jquery.fancybox.css - It used for fancybox
  6. jquery.countdown.css -It is used to set countdown styles.
  7. jquery.selectbox.css - it is used for styleable select box.
  8. loader.css - It used for page loader styles.
  9. docs.css - This style sheet contains general styling for the template structure. It is the base styles color, text styles, header and footer
